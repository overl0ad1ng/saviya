这篇文章我写的就比较懒散，首先什么是 ToDay。这款软件可能有些重名，中文名我想的是：**今是**。

这个名字是 Ai 给我取的，我觉得还不错，取自陶渊明的《归去来兮辞》中的 “觉今是而昨非”。这首诗辞我简单看了一下，但是我觉得很有意义。

或许我也是正该如此，早早就应意识到自己不应该再去想要等到一切，我们的痛苦往往都源于为了满足身体的欲望而忘记了我们真正热爱的。
当我意识到了错误的存在，此刻转身便是正确。

写这款软件的目的其实是为了我自己。我的内心非常冲突，一半是万念俱灰的老翁，一半是意气风发的少年，我也曾想猛猛的干上一波大的，但是很多时候感觉自己像是枯竭了的煤油灯，
想要再散发出一丝光芒都早已艰难。

其实这样的软件我不只一次想要做出来，一个纯粹的为了记录自己、为了自己变得更好为目标的软件。

市面上的软件我并非没有用过，我要么总是忘记、要么提醒太弱、要么功能太少、要么广告满天飞。总之，我想找到一个解决方案，能够解决我存在的所有痛点，于是，ToDay 诞生了。
